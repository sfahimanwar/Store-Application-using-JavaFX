This project is an application that simulates a store application found on the web which allows users to add and remove from carts, buy products etc. The functionality of the store is implemented using OOP principles. A UI for the store is created using JavaFX and follows the MVC paradigm. 

To run the program JavaFX program needs to be installed in the environment. 

JavaFX SDK inhalation instructions - 

1. You must have Java installed first.
2. Download the JavaFX SDK for your operating system from https://gluonhq.com/products/javafx/
3. The downloaded file should be a .zip file. Extract its contents to a location on your hard-drive. Remember where you put it.
4. Go to https://openjfx.io/openjfx-docs/ and select JavaFX and IntelliJ from the menu. Follow the 5 steps for creating a non-modular project through the IDE. Wherever these steps refer to /path/to/javafx-sdk-13, enter the path to the directory where you extracted the files to in the previous step.
